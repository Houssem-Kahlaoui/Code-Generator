# 🔐 Random Password Generator
Un générateur de mots de passe aléatoires simple et élégant, créé avec HTML, CSS et JavaScript.

## ✨ Fonctionnalités

- ✅ Génération de mots de passe aléatoires de 8 caractères
- ✅ Caractères disponibles : majuscules, minuscules, chiffres et symboles
- ✅ Champ en lecture seule (impossible d'écrire manuellement)
- ✅ Copie en un clic dans le presse-papier
- ✅ Design moderne et responsive
- ✅ Notification de copie (personnalisable)
- ✅ Génération automatique au chargement de la page

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
